# **ğŸ“Œ 2. Create a `Gruntfile.js`**
This is where all **Grunt tasks** are configured.

### **Example: A Basic Grunt Task**
Create a file called **`Gruntfile.js`** and add:
```js
module.exports = function (grunt) {
  // Configuration
  grunt.initConfig({
    pkg: grunt.file.readJSON("package.json"),
    
    // Example task: Minify JavaScript
    uglify: {
      build: {
        src: "src/app.js",
        dest: "dist/app.min.js",
      },
    },
  });

  // Load Plugins
  grunt.loadNpmTasks("grunt-contrib-uglify");

  // Register Default Task
  grunt.registerTask("default", ["uglify"]);
};
```
âœ… **This minifies `src/app.js` into `dist/app.min.js`.**
